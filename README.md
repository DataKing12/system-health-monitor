# System Health Monitor (Linux)

A beginner-friendly DevOps project that monitors system health using Bash.

## Features
- Disk usage monitoring
- Warning if disk usage exceeds 80%
- Designed to run via cron
- Built with Bash on Linux

## Technologies Used
- Linux (Ubuntu)
- Bash scripting
- Git & GitHub
- Cron jobs

## How to Run
```bash
chmod +x system_health.sh
./system_health.sh
